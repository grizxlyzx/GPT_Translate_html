安装 Sphinx¶

概览¶

Sphinx 是用 Python 编写的，支持 Python 3.9+。它建立在许多第三方库的基础上，例如安装 Sphinx 时会安装 Docutils 和 Jinja。

Linux¶

Debian/Ubuntu¶

使用python3-sphinx安装apt-get：

$ apt-get install python3-sphinx

如果尚未安装 Python，这将为您安装 Python。

RHEL, CentOS¶

使用 yum 安装 python-sphinx：

$ yum install python-sphinx

如果尚未安装 Python，这将为您安装 Python。

其他发行版¶

大多数 Linux 发行版的软件仓库中都有 Sphinx。通常包名为 python3-sphinx、python-sphinx 或 sphinx。请注意，至少有两个其他包含 sphinx 名称的软件包：一个语音识别工具包（CMU Sphinx）和一个全文搜索数据库（Sphinx 搜索）。

macOS¶

可以使用 Homebrew、MacPorts 或作为 Anaconda 等 Python 发行版的一部分来安装 Sphinx。

Homebrew¶

$ brew install sphinx-doc

更多信息，请参考软件包概览。

MacPorts¶

使用python3x-sphinx安装port：

$ sudo port install py39-sphinx

要设置可执行路径，请使用 port select 命令：

$ sudo port select --set python python39
$ sudo port select --set sphinx py39-sphinx

更多信息，请参考软件包概览。

Anaconda¶

$ conda install sphinx

Windows¶

可以使用 Chocolatey 安装 Sphinx 或手动安装。

Chocolatey¶

$ choco install sphinx

在运行此命令之前，您需要安装 Chocolatey。

更多信息，请参考chocolatey 页面。

其他方法¶

大多数 Windows 用户默认情况下没有安装 Python，因此我们从安装 Python 本身开始。要检查是否已经安装了 Python，请打开命令提示符（⊞Win-r 并输入 cmd）。打开命令提示符后，输入 python --version 并按 Enter。如果安装了 Python，您将看到打印到屏幕上的 Python 版本。如果您尚未安装 Python，请参考 Hitchhikers Guide to Python 的 Python on Windows 安装指南。您必须安装 Python 3。

安装了 Python 后，您可以使用 pip 安装 Sphinx。有关更多信息，请参考下面的 pip 安装说明。

从 PyPI 安装¶

Sphinx 包在 Python 包索引上发布。从 PyPI 安装包的首选工具是 pip。所有现代版本的 Python 都提供了这个工具。

在 Linux 或 MacOS 上，您应该打开终端并运行以下命令。

$ pip install -U sphinx

在 Windows 上，您应该打开命令提示符（⊞Win-r 并输入cmd）并运行相同的命令。

C:\> pip install -U sphinx

安装完成后，在命令提示符上输入 sphinx-build --version。如果一切正常，您将看到您刚刚安装的 Sphinx 包的版本号。

从 PyPI 安装还允许您安装最新的开发版本。通常您不需要（或不想）这样做，但如果您在最新稳定版本中发现了可能的错误，这可能会很有用。要做到这一点，请使用--pre 标志。

$ pip install -U --pre sphinx

使用虚拟环境¶

使用 pip 安装 Sphinx 时，强烈建议使用虚拟环境，这些环境将安装的包与系统包隔离，从而无需使用管理员权限。要在.venv目录中创建一个虚拟环境，请使用以下命令。

$ python -m venv .venv

另见

venv – 创建虚拟环境

警告

请注意，在某些 Linux 发行版中，例如 Debian 和 Ubuntu，这可能需要额外的安装步骤，如下所示。

$ apt-get install python3-venv

Docker¶

Sphinx 的 Docker 镜像发布在 Docker Hub 上。有两种类型的镜像：

  • sphinxdoc/sphinx

  • sphinxdoc/sphinx-latexpdf

前者用于 Sphinx 的标准使用，后者主要用于使用 LaTeX 构建 PDF。请根据您的需求选择一个。

注意

sphinxdoc/sphinx-latexpdf 包含 TeXLive 包。因此镜像非常大（超过 2GB！）。

提示

使用 docker 镜像时，请使用 docker run 命令来调用 sphinx 命令。例如，您可以使用以下命令创建一个 Sphinx 项目：

$ docker run -it --rm -v /path/to/document:/docs sphinxdoc/sphinx sphinx-quickstart

您可以使用以下命令构建 HTML 文档：

$ docker run --rm -v /path/to/document:/docs sphinxdoc/sphinx make html

有关更多详细信息，请阅读docker 镜像的 README 文件。

从源码安装¶

您可以直接从Git 仓库的克隆中安装 Sphinx。这可以通过克隆仓库并从本地克隆安装，或者直接通过git安装。

$ git clone https://github.com/sphinx-doc/sphinx
$ cd sphinx
$ pip install .

$ pip install git+https://github.com/sphinx-doc/sphinx

您还可以下载 Git 仓库的快照，格式为 tar.gz 或 zip。下载并解压后，可以像上面一样使用 pip 安装。
